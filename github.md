# 如何进行文档协作

&emsp;&emsp;我们的FAQs文档基于github来部署，进行文档协作可由以下2种方式进行

1. 先将项目文件fork，然后在自己的版本库中修改后提出pull requests 
2. 由群主将你的github帐号设置为内部协作者，然后直接在当前github工作分支中进行操作，均有以下限制：
   1. master分支为保护分支，不能直接合并
   2. 需要在工作分支working中修改，然后又具有权限的人员进行合并
